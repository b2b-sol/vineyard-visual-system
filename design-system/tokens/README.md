# Vineyard token package

`vineyard.tokens.json` is the canonical DTCG-compatible source. `vineyard.css` is the browser-consumable projection used by the atlas walking slice.

The palette is semantic rather than decorative: vine green represents healthy progress, sun ochre represents time-sensitive attention, harvest red represents blocked or unsafe work, and sky teal identifies synchronization and information states. Every state also carries an icon and text label so color is never the only signal.

Token groups use stable paths intended to migrate directly into Figma variables. Component-specific values should alias these foundations instead of adding one-off colors or spacing.
