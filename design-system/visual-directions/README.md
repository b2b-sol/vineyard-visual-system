# Visual directions

This directory preserves the evidence and decision that established the Vineyard Visual System foundation.

- `moodboard-spec.json` and `exploration/` contain the saved image-only territory exploration created through the Creative Production moodboard workflow.
- `directions.json` is the machine-readable score, acceptance, trace, and selection contract.
- The canonical product comparison is the React route `/visual-directions` and its nine lighthouse routes.
- `validation/visual-direction-review.md` is the human-readable adjudication.
- `validation/visual-direction-renders/` contains exact viewport review evidence.

The selected direction is **Canopy Signal**, refined into a **sunlit fieldbook**. Field Ledger contributes parcel geometry and record-lineage grammar. Vintage Control Room remains contextual inspiration for intake and low-light operations. Moodboard images are reference evidence; they are not canonical interface assets.
